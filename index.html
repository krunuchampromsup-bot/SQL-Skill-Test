การสร้างตารางเสมือน (CREATE VIEW)

CREATE VIEW view_name AS
SELECT column1, column2 FROM table_name WHERE condition;

ตัวอย่าง: สร้าง View ชื่อ it_staff เพื่อดูเฉพาะพนักงานแผนก IT

namedept
SomsakIT
WichaiIT

การเพิ่มข้อมูล (INSERT VIEW)

INSERT INTO view_name (columns) VALUES (values);

ข้อมูลจะถูกส่งไปยังตารางหลัก (Base Table) โดยอัตโนมัติ

การแก้ไขข้อมูล (UPDATE VIEW)

UPDATE view_name SET col = val WHERE condition;

การลบข้อมูล/ตาราง (DELETE/DROP)

DELETE FROM view_name WHERE condition;
DROP VIEW view_name;

โจทย์: จงสร้าง VIEW ชื่อ active_users จากตาราง users โดยเลือกทุกคอลัมน์ที่ status = 'active'

โจทย์: จงเขียนคำสั่งเรียกดูข้อมูล ทั้งหมด จาก VIEW ที่ชื่อ active_users

โจทย์: จงสร้าง VIEW ชื่อ high_price จากตาราง products เลือกทุกคอลัมน์ที่ price > 1000

โจทย์: จงเพิ่มข้อมูลชื่อ 'Notebook' ราคา 25000 ลงใน VIEW high_price (สมมติมีคอลัมน์ name, price)

โจทย์: จงแก้ไขราคา (price) ใน VIEW high_price เป็น 1500 สำหรับสินค้าที่มี id = 5

โจทย์: จงลบข้อมูลออกจาก VIEW active_users สำหรับผู้ใช้ที่มี id = 10

โจทย์: หากต้องการลบตารางเสมือน active_users ออกจากฐานข้อมูล ต้องใช้คำสั่งใด

โจทย์: จงสร้าง VIEW ชื่อ staff_list โดยเลือกเฉพาะคอลัมน์ name จากตาราง employees

โจทย์: จงเขียนคำสั่ง SELECT ข้อมูลจาก VIEW staff_list โดยเรียงชื่อตามตัวอักษร

โจทย์: จงใช้คำสั่งลบตารางเสมือน high_price ออกจากระบบฐานข้อมูล
